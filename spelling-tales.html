Spelling Tales

Game App

The UI designs (below) were originally created for a module on a master's program. It was later extended into a personal project. I created the UI designs for the visual design module on the MA Digital Media Design 2022 program. We were required to create 3 UI designs for a website or mobile application.

I chose to design for a mobile game. The title of the game is Spelling Tales, and the target audience is young children. The game helps children learn how to spell through storytelling.

I wanted to make use of the Adobe Creative Suite to create visual elements from scratch. But I also loved the idea of delving into the world of digital games and experimenting with layout, typography and colour to design my own game.

At the moment, these are purely designs but there is potential to create a prototype and perhaps an actual app that children can interact and play with.
